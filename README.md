# 🗨 ひとこと掲示板アプリ

## 📌 アプリ概要

ユーザーが名前と短いメッセージを投稿し、一覧表示できる掲示板アプリです。
投稿・表示を通して、Webアプリケーションの基礎構造（MVCモデル）を学習

### 主な特徴

* MVCモデル構成
* DAOパターンによるDB操作分離
* 投稿機能
* 一覧表示
* 入力値チェック
* 拡張可能な設計

---

## 🧩 システム構成

```
[ ユーザー ]
      ↓
[ ブラウザ ]
      ↓
[ JSP（View） ] ← 表示
      ↑
[ サーブレット（Controller） ]
      ↓
[ DAO（Model） ]
      ↓
[ MySQL データベース ]
```

---

## 🗃 データベース設計

### テーブル名：MUTTER

| カラム名       | データ型         | 説明             |
| ---------- | ------------ | -------------- |
| id         | INT          | 投稿ID（主キー・自動採番） |
| name       | VARCHAR(100) | 投稿者名           |
| text       | VARCHAR(255) | 投稿内容           |
| created_at | TIMESTAMP    | 投稿日時（自動設定）     |

### 設計意図

* 各投稿を一意に識別するためIDを設定
* 表示順制御のため日時を保存
* 入力文字数を制限しDB負荷を回避

## 🖥 画面設計

### メイン画面（main.jsp）

構成要素：

* タイトルエリア
* 投稿フォーム

  * 本文入力欄
  * 投稿ボタン
* 投稿一覧エリア

  * 投稿日時

### 画面フロー

```
アクセス
   ↓
つぶやき一覧表示
   ↓
入力
   ↓
投稿
   ↓
再表示
```

## ⚙ 機能設計

### Controller（MainServlet）

| メソッド   | 処理内容           |
| ------ | -------------- |
| doGet  | すべての投稿を取得し一覧表示 |
| doPost | 投稿内容を受け取りDBへ登録 |

### Model（MutterDAO）

| メソッド      | 処理内容  |
| --------- | ----- |
| findAll() | 全投稿取得 |
| create()  | 投稿登録  |

---

## ✅ 入力バリデーション仕様

| 項目 | ルール            |
| -- | -------------- |
| 名前 | 空白不可 / 最大100文字 |
| 本文 | 空白不可 / 最大255文字 |

エラー時は画面上にメッセージ表示。
